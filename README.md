# Typing Speed Simulator

A command-line typing test that measures typing speed and accuracy while providing autocorrect suggestions and multiplayer race support.

## Features
- Words-per-minute (WPM) and accuracy calculation
- Autocorrect using edit distance algorithms
- Paragraph selection by topic
- Multiplayer typing race tracking progress in real time

## Concepts Demonstrated
- Recursion and higher-order functions
- String processing algorithms
- Edit distance (Levenshtein-style diff)
- Functional decomposition and modular design
